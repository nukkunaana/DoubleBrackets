Double brackets
=============

### Сайт трехстраничник.


#### Функционал:

- слайдер промо и отзывов,
- карусель логотипов,
- плавный переход по категориям меню блюд,
- кнопка выбора пиццы,
- Гугл карта.
 
#### Макеты:
Находятся в папке "layouts".
** Шрифт изменён.**
