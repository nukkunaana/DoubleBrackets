Double brackets
=============

### Сайт трехстраничник.


#### Функционал:

- слайдер промо и отзывов,
- карусель логотипов,
- плавный переход по категориям меню блюд,
- кнопка выбора пиццы,
- модальное окно, при клике на "Акции",
- Гугл карта,
- проверка валидности форм.
 
#### Макеты:
Находятся в папке "layouts".
